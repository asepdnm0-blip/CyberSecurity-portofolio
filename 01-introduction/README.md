# Introdcution

Saya adalah mahasiswa [Jurusan] di [Universitas] dengan minat
di bidang offensive security dan malware analysis. Portofolio

## Methodology
Pengujian menggunakan PTES (Penetration Testing Execution Standard)
dikombinasikan dengan MITRE ATT&CK; framework untuk mapping teknik.

## Ethics Statement
Semua aktivitas dalam portofolio ini dilakukan di lingkungan lab
virtual yang terisolasi dari jaringan nyata. Tidak ada sistem milik
pihak lain yang diuji. Pengetahuan ini digunakan semata-mata untuk
keperluan pembelajaran defensive security.
